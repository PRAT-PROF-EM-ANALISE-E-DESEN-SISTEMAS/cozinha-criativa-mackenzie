1- Criar nova branch derivando ela  'from main' (Branch main remota)

2- Edita o codigo/Atualiza o código

3- Commmita as alteraçõe do código (Comentário)

4- Faz o 'push' dos commits feitos (Então sua branch vai pro github)

5- Criar PR (Pull Request) no github com a 'branch nova' apontando para branch principal main 'to main'

6- Após aprovação do time merge pull request do codigo novo para o código principal
