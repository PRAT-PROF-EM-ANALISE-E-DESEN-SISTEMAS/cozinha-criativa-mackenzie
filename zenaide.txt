- Criar nova branch derivando ela  'from main' (Branch main remota)

- Edita o codigo/Atualiza o código

- Commmita as alteraçõe do código (Comentário)

- Faz o 'push' dos commits feitos (Então sua branch vai pro github)

- Criar PR (Pull Request) no github com a 'branch nova' apontando para branch principal main 'to main'

- Após aprovação do time merge pull request do codigo novo para o código principal
